# Execute on development

1. Install Dependecies

```
npm i
```

2. Install NEST CLI

```
npm i -g @nestjs/cli
```

3. Database up (Mongo)

```
docker compose up -d
```

3. Seed database

```/hash
# endpoint
http://localhost:PORT/seed
```

## Stack

- Mongo DB
- Nest
